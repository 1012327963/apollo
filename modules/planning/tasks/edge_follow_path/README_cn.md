# Edge Follow Path

该任务根据参考线生成紧贴车道右侧边缘的轨迹，用于 `EdgeFollowScenario` 场景。
在计算右侧边界时会调用 `ReferenceLine::GetRoadWidth` 读取高精地图中记录的
`road_width`，确保车辆沿道路外沿行驶而不是仅贴近车道线。

在生成路径时，如果检测到贴边处存在静态或长时间静止的障碍物，会在障碍物前逐渐偏向车道中心，在通过障碍物后 10 米内回到贴边位置。

任务开始后，会在 `initial_shift_length` 米距离内从车辆当前位置平滑过渡到
车道右侧边缘，整个过程车辆朝向始终与车道方向一致。
